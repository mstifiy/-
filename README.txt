《大家来找茬》游戏辅助工具
使用python开发
关键库：pywin32、opencv-python、wxpython

使用步骤：
（1）下载python（python3），导入相关必要库。
（2）运行源码（main.py）
（3）弹出GUI界面，打开“大家来找茬”游戏界面并进入匹配房间（在没有打开游戏界面时切忌不要点击“运行”按键！！！）
（4）使用时，当游戏界面中出现两张图片时，鼠标左击界面中间的“运行”按键，辅助信息将会以图片的形式出现在按键上下方。
    上方为轮廓差异的GIF动态图，下方为颜色差异的JPG图片。在找不同时应当首先找出GIF图中的提示信息，在GIF提示信息
    不足以找出5个不同之处时，需参考颜色差异图，完成剩余不同点的寻找。
